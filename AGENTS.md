# 전역 Claude 규칙

**Always respond in Korean**

## 응답 스타일
- 짧고 실용적으로, 불필요한 서론/요약 생략
- 모든 응답 마지막에 실제로 참고한 규칙 파일을 **빠짐없이** 프로젝트 기준 상대경로로 나열
- **규칙 문서뿐 아니라 작업 중 직접 읽은 파일도 모두 포함**

예시:

```text
📋 참고한 규칙: AGENTS.md, README.md, src/utils/projection.ts
```

## 코딩 컨벤션
- 변수/함수명: camelCase
- 인덴트: 4칸 스페이스
- 주석: 한국어, 코드(변수/함수명): 영어
- 약어 지양, 명확한 의도 표현

## 작업 규칙
- 코드가 수정되면 작업 종료 전 반드시 `재빌드`를 수행한다.
- 코드가 수정되면 작업 종료 전 반드시 웹서버 배포 경로에도 반영한다.
- 이 프로젝트의 기본 웹서버 반영 경로는 `/var/www/fire_calc` 이다.
- 코드 수정 후 웹에서 확인 가능한 변경이라면 가능하면 실제 서비스 URL에서도 반영 여부를 확인한다.
- 계산식이 수정되면 관련 설명 문구, README, 프롬프트 내용 중 영향받는 부분이 있으면 함께 수정한다.
- 프롬프트 생성 로직이 계산 규칙을 참조한다면, 계산식 변경 시 프롬프트 내용도 함께 최신 규칙으로 맞춘다.
- UI만 수정하더라도 그 변경이 프롬프트 노출 방식이나 입력 해석에 영향을 주면 프롬프트 내용도 같이 점검한다.
